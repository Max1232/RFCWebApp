# RFCWebApp

The original intake-tracker PWA prototype for the Roslindale Food Collective has been scrapped.

This repo now just hosts a single static page (`index.html`) — a "click to email" redirect for the Centre/Walter Streets traffic safety advocacy effort. It's a pre-filled `mailto:` link (recipients + subject + letter body) behind a button, served via GitHub Pages at `https://mcoss0.github.io/RFCWebApp/`, and linked to from a QR code so people can send the letter with one tap.
